<div class="tutorial-steps">

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">1</div>
<div class="tutorial-step__content">
<h3>Tap "Host Event" on the home screen</h3>
<p>From the Marker home screen, tap the <strong>Host Event</strong> button. This starts the event creation flow. If you don't see it, make sure you're signed in — the button is only visible to signed-in users.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">2</div>
<div class="tutorial-step__content">
<h3>Select your course</h3>
<p>Search for your course by name or browse nearby courses. Marker has a large database of courses with tee and hole data pre-loaded. Select the correct course from the results — this sets the par values and yardages for your scorecards.</p>
<p>If your course isn't listed, you can add it manually with a custom name and hole-by-hole par configuration.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">3</div>
<div class="tutorial-step__content">
<h3>Name your event</h3>
<p>Give the event a name — something your group will recognize. It can be as simple as "Saturday Round" or as specific as "Club Championship 2026." The name appears on scorecards, the leaderboard, and settlement screens.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">4</div>
<div class="tutorial-step__content">
<h3>Set the date</h3>
<p>Pick the date of your round. This is used for record-keeping, series tracking, and USGA posting if you enable it. It defaults to today but you can adjust it for future or past events.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">5</div>
<div class="tutorial-step__content">
<h3>Save the event</h3>
<p>Tap <strong>Save</strong> or <strong>Create Event</strong>. Your event is created and you'll be taken to the event dashboard. From here you can add players, create scorecards, and configure games — in any order.</p>
</div>
</div>

</div>

<div class="tutorial-note">
<p><strong>Multi-group events:</strong> If you're running a tournament with multiple groups teeing off at the same time (shotgun) or in a wave start, create one event and add all players before creating scorecards. Marker keeps everyone on the same leaderboard automatically.</p>
</div>
