<div class="tutorial-steps">

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">1</div>
<div class="tutorial-step__content">
<h3>Open the Scorecards section</h3>
<p>From the event dashboard, tap the <strong>+</strong> button next to <strong>Scorecards</strong>. Add all players to the event before creating scorecards — players are assigned to groups after the scorecards are created.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">2</div>
<div class="tutorial-step__content">
<h3>Set the start time and number of groups</h3>
<p>Enter the start time for the first group and choose how many groups to create. For a single group, enter 1. For a larger field, enter the total number of groups — one scorecard per group.</p>
<p>Set the starting hole if your round begins on a hole other than hole 1.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">3</div>
<div class="tutorial-step__content">
<h3>For multiple groups — choose Tee Times or Shotgun Start</h3>
<p>When creating more than one group, choose the start format:</p>
<ul>
<li><strong>Tee Times</strong> — groups start sequentially from hole 1. Set the spacing between groups (typically 8–10 minutes). Marker assigns each group a start time: the first group at your chosen time, then each subsequent group delayed by the interval.</li>
<li><strong>Shotgun Start</strong> — all groups start at the same time from different holes. Marker assigns each group a starting hole and distributes them across the course.</li>
</ul>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">4</div>
<div class="tutorial-step__content">
<h3>Assign players to groups</h3>
<p>After scorecards are created, tap any group from the event dashboard to open it and add players. Select up to 5 players per group from the event's player roster. Players are matched to groups based on tee compatibility.</p>
<p>You can reassign players to different groups at any time before the round begins.</p>
</div>
</div>

</div>

<div class="tutorial-note">
<p><strong>Tip:</strong> Create your scorecards before adding games. Event-wide games draw their player list from the full event roster, while scorecard-level games are added from within individual scorecards after groups are set.</p>
</div>
