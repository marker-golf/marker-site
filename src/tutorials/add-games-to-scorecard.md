<p>With players set, the last step before scoring is choosing what to play. Marker runs multiple games simultaneously from a single set of entered scores — your group enters scores once and Marker calculates results for every format you've configured.</p>

<div class="tutorial-steps">

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">1</div>
<div class="tutorial-step__content">
<h3>Tap + next to Games</h3>
<p>From the Scorecard screen, tap <strong>+</strong> in the Games section header. This opens the Add game screen.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">2</div>
<div class="tutorial-step__content">
<h3>Choose a game</h3>
<p>The <strong>Select Game</strong> screen lists all available formats alphabetically. Each entry shows a one-line description so you can identify the format at a glance. Games labeled <strong>Premium</strong> require a Marker premium subscription.</p>
<p>Marker supports eleven formats in the current release: High-Low, Match Play, Nassau, Nines (5-3-1), Scotch, Skins, Stableford, Stroke Play, Team Stableford, Team Stroke Play, and Wolf.</p>
<p>For rules, scoring examples, and strategy for each format, see the <a href="/games/">Games guide library</a>. This tutorial explains how to configure them in Marker — not how to play them.</p>
<p>Tap a game to open its configuration form.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">3</div>
<div class="tutorial-step__content">
<h3>Configure the game</h3>
<p>Every game form is organized in the same order: <strong>Handicap</strong> settings first, then the <strong>Players</strong> checklist, then any game-specific settings below.</p>
<p>All players on the scorecard are pre-selected. Uncheck any player who shouldn't be in this game. Each player row shows their tee and course handicap. When net scoring is on, their playing handicap and stroke-off-low-ball strokes are shown as well.</p>
<p>Configuration options vary by format. Not every game exposes every setting.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">4</div>
<div class="tutorial-step__content">
<h3>Set handicaps</h3>
<p>The Handicap section appears at the top of every game form. Toggle <strong>Net</strong> to apply course handicaps, or <strong>Gross</strong> to ignore them.</p>
<p>Net mode adds one option immediately: <strong>Stroke off the low ball</strong>, which allocates strokes relative to the lowest-handicap player in the game rather than course par.</p>
<p>Tap <strong>show advanced</strong> for additional controls:</p>
<ul>
<li><strong>Handicap Allowance</strong> — percentage of each player's course handicap to apply. Default is 100%.</li>
<li><strong>Limit Playing Handicap</strong> — caps strokes at a maximum number.</li>
<li><strong>Use half strokes</strong> — rounds half-stroke allowances down.</li>
<li><strong>No strokes on par 3's</strong> — removes handicap stroke allocation from par 3 holes.</li>
<li><strong>Override player handicaps</strong> — set a manual handicap for individual players in this game only, without changing their profile.</li>
</ul>
<p>For a full explanation of course handicap calculation, see <a href="/tutorials/handicap-settings/">Using Handicaps in Marker</a>.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">5</div>
<div class="tutorial-step__content">
<h3>Set the accounting</h3>
<p>Accounting fields appear in the game-specific settings section below the player list. They vary by format.</p>
<p>The amount field is always optional. Leave it blank to track the game with no money involved — Marker calculates results regardless.</p>
<p>Common accounting patterns:</p>
<ul>
<li><strong>Skins</strong> — choose <strong>Pool buy-in</strong> (players contribute to a pot; winners split by skins won) or <strong>Per-skin</strong> (each player pays the winner per skin). Enter the dollar amount for either mode.</li>
<li><strong>Match games</strong> (Nassau, Match Play) — accounting is set per match within the game.</li>
<li><strong>Team games</strong> (Team Stroke Play, Team Stableford) — accounting is set per sub-total segment; configure teams first, then set the amount for the full round or any sub-round.</li>
<li><strong>Most other formats</strong> — a single pool buy-in amount that winners split at settlement.</li>
</ul>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">6</div>
<div class="tutorial-step__content">
<h3>Save the game</h3>
<p>Tap <strong>Save game</strong>. Marker returns you to the Scorecard screen, where the new game appears in the Games list showing the format name and player count.</p>
<p>To add another game, tap <strong>+</strong> again and repeat the process. There's no limit on the number of games per scorecard. Your group could run Skins and Nassau at the same time — you enter scores once and Marker calculates both.</p>
</div>
</div>

<div class="tutorial-step">
<div class="tutorial-step__marker" aria-hidden="true">7</div>
<div class="tutorial-step__content">
<h3>Review your setup</h3>
<p>Before tapping <strong>View scorecard</strong>, confirm:</p>
<ul>
<li>The correct players are listed</li>
<li>The correct games are listed</li>
<li>Teams are assigned correctly for team formats</li>
<li>Handicap settings match how your group plays</li>
<li>Accounting amounts are correct</li>
</ul>
<p>To change any game, tap it in the Games list to open the Edit game screen. Update settings and tap <strong>Save changes</strong>, or tap <strong>Remove Game</strong> at the bottom to delete it. Games can be edited until the scorecard is locked.</p>
<p>When everything looks right, tap <strong>View scorecard</strong> to move into scoring.</p>
</div>
</div>

</div>

<div class="tutorial-note">
<p><strong>Need a custom format?</strong> If none of the standard formats match how your group plays, Marker's AI game generation can help you build a custom game. See <a href="/tutorials/ai-game-generation/">Create a Custom Game with AI</a>.</p>
</div>
